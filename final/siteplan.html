Site Plan

Site Name:

The Game Helps: Civilization 7(It represents a game forum/wiki specifically talking about the game Civilization 7) domain: helps-civ.org

Site Purpose:

The site provides a center for fans to talk about the game by providing users with articles explaining aspects of the game as well as information on the game itself.

Scenarios:

Where is the best place to start for biginners? How can I write an article?

Color Schema:

Blue (39, 39, 121) blue will be a background color of the paragraphs, as well as the color of the headings. Burlywood (222, 184, 135) burlywood will be the color of the paragraphs in the page as well as the background of the headings.

Typography:

The headings will be "Roboto Condensed" sans-serif and the paragraphs will be "Playfair Display" serif.
